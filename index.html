Support de Cours Azure

1. Mise en réseau avec Azure VNET

Créer un réseau privé Azure, connecter des ressources, configurer des sous-réseaux.

az network vnet create ...

2. Sécurité dans Azure

Utilisation des NSG, RBAC, Azure AD, Key Vault pour protéger les ressources cloud.

az network nsg create ...

3. DevOps avec Bicep

Déploiement d'infrastructure via du code déclaratif lisible et modulaire.

az deployment sub create --template-file main.bicep
